Book Bank

Screenshots

Description

BookBank is an application that is used to upload PDF Books & Book Ads online which are published once reviewed by the admin. The application uses Collaborative Algorithm (Item-to-Item) to filter out items and display them under Recommendation section. The user can also see Books category-wise which are fetched from firebase server. The user can download PDF books on their phone and read them. The user can also purchase a book from some other user by contacted them through Book Ads. App UI is designed using Material Design to showcase an eye-catching application.
