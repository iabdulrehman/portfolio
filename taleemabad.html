Description

Taleemabad Learning app is an Educational app designed for Nursery to Grade 6 students. User can access app after Authentication and watch videos which are streamed through Rapid server. Videos can be downloaded which are encrypted for offline use through app only. Users can take 14 different types of tests within the app and also practice to learn some topic or subject. The app is designed to support different screen sizes and optimized for different devices. The app uses Retrofit to make network calls and get the latest data through the API which are hosted on Cloud. The app uses Room Database to store chapter, topic and some other useful data offline for faster app experience until the data is updated on Cloud. The App contains some games which are built in native Java using Canvas and Services.
