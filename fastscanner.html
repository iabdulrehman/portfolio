Screenshots

Description

Fast Scanner App is used to scan & generate pdf of any important documents. The App uses Android Camera2 API to take pictures and uses OpenCV SDK for Edge Detection & Image effects. User can create single Image or Multi-image documents from within the app. PDF file or JPGs of any document can be easily shared with others.
