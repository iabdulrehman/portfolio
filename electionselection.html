Description

Election Selection App is an election survey based application. User can cast vote for favourite candidate by filling in required information. Phone Authentication is performed using custom phone API. The application uses Oracle 11g as backend to store relevant data using Rest APIs. User can see top 5 candidate results in a list and analyze data using the Bar Chart & Pie Chart.
